    Voyage : A Firefox Addon to Rediscover Your Web Browsing History

    Rediscover your web browsing history in the timeline: websites, photos, videos, and even tweets.

    Get Voyage from Mozilla Add-ons

    Mozilla Firefox is required to use the add-on. Compatibility: 3.5 - 10.*

    • Check the manual for details about the add-on.
    • For developers, check and fork the source code on GitHub under GPLv3.
    • Voyage is one of the Grand Prize Winners for Best New Add-on in the Extend Firefox 3.5 Contest.
