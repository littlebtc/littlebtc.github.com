    Hello Octopress!

    收假前半小時，終於讓新網站上線了！ XDDDD

    隨著 Octopress 在 Web Developer 間逐漸流行起來，寫過 Rails 的我也開始對這個基於 Ruby 的網誌系統產生興趣。

    經過一番研究之後，決定改用 Octopress 來寫網誌，並且善用其中 Jekyll 的功能，把自己過去用 DZSlides 寫的，和未來想做的 HTML5 投影片也一起放進同一個網站裡。這樣自己的網站內容都可以統一保管在 GitHub Pages 上面了 :)

    考慮到轉移太麻煩，我也選擇使用砍掉重練的方式。為了避免舊新網誌網址混淆，決定把整個 http://blog.littleb.tc 拿掉，舊的移回 http://littlebtc.blogspot.com，新的網誌放在 http://littleb.tc/blog 裡頭，並且用 WM 的 VPS 把舊網誌的舊文章轉到舊網址。

    這個網站使用了 SkyArrow 製作的 Slash 佈景主題，簡潔而漂亮 :)

    這個 Octopress 將會是我的個人網站、網誌與投影片的放置點。請多指教！
