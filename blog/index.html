    夏天結束與 NicoFox 四週年之時

    不知不覺間已經退伍兩個半月了，而夏天也隨著東北季風的到來結束了。

    因此，又到了要稍微反省一下自己做了什麼的時間。

    這個夏天大半寫程式的時間都放在工作上吧。有了新的 Startup 工作，作出了一些非常棒的成品，希望可以趕快跟大家見面。在個人專案方面，NicoFox 推出了內部改寫和針對 Zero 修正的 1.0 Beta 系列，TouchLimit 在八月期間進行了最後的 MongoDB 微調（還因此講了 COSCUP！）

    個人生活方面，利用難得留在台中的機會耍了很多 Troll，雖然感覺最後什麼都沒得到，但也是一段有趣的經驗呢。七月瘋狂南部行、考到了駕照、歡送要去美國奮鬥的 WM、跟大飛籌辦了 NicoFox 傳說中的 FF20 胸章計畫，而八月自己跑去聽 Cytus Live、COSCUP 2012 還有 Prismatic Music 的硬核 Party，都是很棒的活動！ :)

    也偷偷買了新的隨身 DC 和新的機械式鍵盤，在增加生產力與創造力方面也應該能有很大貢獻（喂）

    展望秋天，NicoFox 1.0、TouchLimit 5.0 都正在默默拖搞之中，還有很多東西會跟大家見面！

    自己真的還不夠，各方面都是。要讓自己更強，才能在這個世界作些什麼。

    Hello Octopress!

    收假前半小時，終於讓新網站上線了！ XDDDD

    隨著 Octopress 在 Web Developer 間逐漸流行起來，寫過 Rails 的我也開始對這個基於 Ruby 的網誌系統產生興趣。

    經過一番研究之後，決定改用 Octopress 來寫網誌，並且善用其中 Jekyll 的功能，把自己過去用 DZSlides 寫的，和未來想做的 HTML5 投影片也一起放進同一個網站裡。這樣自己的網站內容都可以統一保管在 GitHub Pages 上面了 :)

    考慮到轉移太麻煩，我也選擇使用砍掉重練的方式。為了避免舊新網誌網址混淆，決定把整個 http://blog.littleb.tc 拿掉，舊的移回 http://littlebtc.blogspot.com，新的網誌放在 http://littleb.tc/blog 裡頭，並且用 WM 的 VPS 把舊網誌的舊文章轉到舊網址。

    這個網站使用了 SkyArrow 製作的 Slash 佈景主題，簡潔而漂亮 :)

    這個 Octopress 將會是我的個人網站、網誌與投影片的放置點。請多指教！
