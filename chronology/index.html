    我的 Web 歷程

    從 1998 在國小教室第一次打開 Microsoft FrontPage 的那一刻起，我成為了一個 Web Developer。 從 IE 5 到 Firefox 10，從 CGI/Perl、PHP 到 Ruby on Rails，長達 14 年的網頁設計冒險之旅，至今尚未停歇。

    1998～2001：初創期

    還是個國小學生的時候，我就開始用 FrontPage 和 PhotoImpact 開始亂寫起網站了。 :P 當時的作品已多不可考。

    2001～2003：M-Stage 時期

    2001 年，以小魔女 DoReMi 為主題做起了個人網站「M-Stage」（已關閉），也因為這個網站開始學習起當時流行的 CGI/Perl，算是我網站開發的萌芽階段。

    2003～2006：PHP 時期

    在「天真的驚直」將 n@board 正體中文化後，我開始把研究的重心移動到 PHP 上。當時以 n@series 為基礎成立了個人網站「S.S.T. Dream House」（已關閉），並撰寫多個基於 n@board 的 Patch 系加強套件。此階段和台中一中同校的吳冠龍以團隊參與國際科展電腦科學科，獲選 Intel ISEF 2005 Finalist。

    2006～2008：維基百科時期

    2006 起因緣際會加入了中文維基百科的編輯，後來成為了中文版管理員（現在已解任）。後來參與了 Wikimania 2007 國際會議的競標和籌備，負責網站和報名系統製作。

    2008～：MozTW 時期

    2008 年下半年，在台大宿舍裡完成了 NicoFox，也因此慢慢加入 MozTW 社群的運作之中。後來靠 Voyage 拿下 Extend Firefox 3.5 大獎，更成為了 Mozilla 的附加元件推廣大使。曾在 COSCUP 和 OpenFoundry 工作坊演講過和附加元件有關的主題。

    2010～：Rails 時期

    2010 年開始學習 Ruby on Rails，參與了 OnionSky 的技術實作，並且因為自己的興趣，架設了 TouchLimit 成績單系統。
