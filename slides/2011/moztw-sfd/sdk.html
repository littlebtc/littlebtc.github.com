什麼是 Add-on SDK？

  • 以前叫做 Jetpack
  • 開發 Firefox 套件的新工具
  • 簡單、容易、問題少

Add-on SDK 套件特色

  • 免重開瀏覽器直接啟用
  • 只需 HTML / CSS / JavaScript
  • AMO 上會自動調整相容性

Add-on SDK 1.1 提供的內容：

  • self
  • windows, tabs, context-menu, hotkeys
  • private-browsing, passwords
  • page-mod, page-worker, request
  • simple-storage, timers
  • notifications, panel, widget
  • clipboard, selection

使用 Add-on SDK 的方式

  • 下載 Add-on SDK
    使用 bash + Python + Firefox 進行開發
  • 使用 Add-on Builder 線上開發

Demo!?

Add-on SDK 目前缺點

  • 不容易提供選項
  • 不容易進行本地化（翻譯）
  • 與傳統形式套件不易混用
  • 很難使用 XUL 更改或新增介面
  • 不支援 Firefox 行動版
  • 這些都在改善之中
